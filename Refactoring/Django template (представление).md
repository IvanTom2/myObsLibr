___
Template - это HTML шаблон, который отправляется в ответ на запрос. В данном шаблоне присутствуют, конечно же, различного рода вставки и генерация данных, например, как в jinja2. 
___
[[Code]]:
```
<html>
{{ item_name }}
</html>
```
